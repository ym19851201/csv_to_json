# 使い方
---------

Excelから出力したCSVを想定しています。

例えばこのエクセルファイルをCSVに出力すると


|        | A        | B            | C        |
|:------:|:---------|:-------------|:-------  |
|1       |name      |skill         |waza1     |
|2       |ガブリアス|さめはだ      |じしん    |
|3       |マッギョ  |せいでんき    |ほうでん  |
|4       |ハッサム  |テクニシャン  |バレパン  |

多分こんなんなります。

```
name,skill,waza1 
ガブリアス,さめはだ,じしん 
マッギョ,せいでんき,ほうでん
ハッサム,テクニシャン,バレパン
```

こんな感じで1行目に列名が来るタイプの表のみ想定しています。

```
$ ruby run.rb
```

実行後、指示に従ってあげてください。
そしたらjson形式で表示してくれます。


また、例では日本語を使っていますが、日本語対応してません。
